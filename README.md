# Redmine Tech Radar

Adds a Tech Radar to Redmine, allowing teams to visualize and manage their preferred technology stack.

### Setup

```sh
git clone https://github.com/renuo/redmine_techradar plugins/redmine_techradar

RAILS_ENV=production NAME=redmine_techradar bin/rails redmine:plugins:migrate
```
